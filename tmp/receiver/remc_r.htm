MACROS (MC)
RECEIVER


!!MCvar:S@name@;Sets up macros. You can use a text name $name$ for variable var. Applicable to all variables.Not applicable for flags
Notes:
First you should set up the macro before first using of macro name:
!#MCv100:S@Var 100@;
Variable v100 has now a second name "Var 100". The length of a macro name is limited by 8 chars. You can use longer names but only first 8 characters will be used to find a variable by name.
You can change a macro name. Then you can use a macro name at any place where you can use a variable. It must be enclosed in '$' signs.
Examples:

Macro name is inside $$ and has a % sign before.
