THE NEW WOG OBJECTS
castle Object List

You can find these objects in WOG Map Editor's castle list. They have been inserted into the editor as empty objects for personal customization.

Note: However, you should know that some objects are already in use by some WOG Team scripts. This means that your custom scripts should avoid those objects, because they might be influenced by team scripts, if you use WoGification. Otherwise you can use any building for any script if you insert/write scripts yourself (do not use Team's scripts).
